# 编译原理实践相关代码
## Lab1
简单的gcc使用，目录中包含了sample.c源代码及编译预处理、编译、汇编、及最终的目标文件。

## Lab2
词法分析器，主要采用自动机的方式实现。

### re2nfa
该目录实现由正则表达式向NFA的转换，输出包含NFA的状态数、边数及具体的转换关系等相关信息，还包括.dot的图片文件以实现更好的可视化。利用make工具可以更方便的进行调试和结果生成（make可生成main执行文件，make nfa.png可利用dot文件生成png文件使得装换图可视化）。

### nfa2dfa
该目录实现由NFA向DFA的转换。

### DFAmin
该目录实现DFA最小化。需要注意的是，该目录继承了上述目录的所有功能，因此在使用时可直接调用该目录，但是所使用的函数方法源代码并不在该目录中。make工具与上述目录中相同

## Lab3
语法分析器，采用LL1文法预测分析（自顶向下）实现。

### leftrecursion_eliminator
该目录消除左递归，输出消除前和消除后的文法

### LeftCommonFactor_Extractor
该目录提取左公因子，输出提取前和提取后的文法

### First_Follow
该目录求解非终结符的First集和Follow集，同时输出算法运行过程日志

### LL1_Grammar
该目录构建预测分析模型，输出预测分析表，同时终端输入待检验字符串，输出是否符合文法的结果
